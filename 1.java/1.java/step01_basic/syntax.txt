package 선언구;

변수 - 데이터

생성자 - 메모리에 확보 가능하게 해주는 구성 요소
	클래스명([..]){[..]} //로직이 있을수도 있고 없을수도 있기때문에 대괄호로 하는거

	반환타입 메소드명 ([..]){[..]}
static{} - 리소스(메모리....) 절약용 효율적인 구성 요소
}
